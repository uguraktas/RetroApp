## التكوين

تعلم كيفية تكوين التطبيق ليناسب احتياجاتك.

### متغيرات البيئة

يستخدم التطبيق متغيرات البيئة للتكوين. قم بإنشاء ملف `.env` في الدليل الجذر:

```bash
# قاعدة البيانات
DATABASE_URL="postgresql://user:password@localhost:5432/mydb"

# المصادقة
AUTH_SECRET="مفتاحك-السري"
AUTH_TRUST_HOST="true"

# تكوين API
NEXT_PUBLIC_API_URL="http://localhost:3001/api"

# علامات الميزات
NEXT_PUBLIC_ENABLE_ANALYTICS="false"
```

### تكوين التطبيق

يوجد التكوين الرئيسي في `packages/config/index.ts`:

```typescript
export const appConfig: AppConfig = {
  appName: "اسم التطبيق الخاص بك",
  description: "مجموعة بداية كاملة",
  author: "اسمك",
  i18n: {
    default: "en",
    locales: {
      en: "English",
      tr: "Türkçe",
      ar: "العربية",
    },
  },
};
```

### التدويل (i18n)

يدعم التطبيق لغات متعددة:

- **اللغة الافتراضية**: الإنجليزية (en)
- **اللغات المدعومة**: التركية (tr)، العربية (ar)

لإضافة لغة جديدة:

1. قم بتحديث التكوين في `packages/config/index.ts`
2. أضف الترجمات في الملفات المناسبة
3. قم بتحديث مطابق الوسيط في `apps/web/src/middleware.ts`

### تكوين السمة

قم بتكوين السمة في مكوناتك:

```typescript
import { ThemeProvider } from "next-themes";

<ThemeProvider
  attribute="class"
  defaultTheme="system"
  enableSystem
>
  {children}
</ThemeProvider>
```

### تكوين البناء

خصص بناء Next.js في `next.config.ts`:

```typescript
const nextConfig: NextConfig = {
  typedRoutes: true,
  transpilePackages: ["shiki", "@repo/config"],
  // أضف التكوين المخصص الخاص بك هنا
};
```
