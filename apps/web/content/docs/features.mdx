## Authentication

CodeBaseHub includes a complete authentication system powered by **Better Auth**, supporting multiple authentication methods and security features.

### Email/Password Authentication

```typescript
import { authClient } from "@/lib/auth-client";

// Sign up
await authClient.signUp.email({
  email: "user@example.com",
  password: "secure-password",
  name: "John Doe",
});

// Sign in
await authClient.signIn.email({
  email: "user@example.com",
  password: "secure-password",
});
```

### OAuth Authentication

```typescript
// Sign in with OAuth providers
await authClient.signIn.social({
  provider: "github",
  callbackURL: "/dashboard",
});

// Supported providers:
// - GitHub
// - Google
// - Discord
```

### Two-Factor Authentication

```typescript
// Enable 2FA
await authClient.twoFactor.enable({
  password: "user-password",
});

// Verify 2FA code
await authClient.twoFactor.verify({
  code: "123456",
});
```

> **Note:** All authentication routes are protected with CSRF tokens and secure session management.

## Database

PostgreSQL database with **Drizzle ORM** for type-safe database queries.

```typescript
import { pgTable, text, timestamp } from "drizzle-orm/pg-core";

export const users = pgTable("users", {
  id: text("id").primaryKey(),
  email: text("email").notNull().unique(),
  name: text("name").notNull(),
  createdAt: timestamp("created_at").defaultNow(),
});
```
