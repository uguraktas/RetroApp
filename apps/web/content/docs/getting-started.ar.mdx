## البدء

مرحبا! سيساعدك هذا الدليل على البدء مع التطبيق.

### المتطلبات الأساسية

قبل أن تبدأ، تأكد من أن لديك:

- Node.js 18.x أو أعلى
- npm أو pnpm مدير الحزم
- معرفة أساسية بـ React و Next.js

### التثبيت

قم بتثبيت التبعيات:

```bash
pnpm install
```

### تشغيل التطبيق

ابدأ خادم التطوير:

```bash
pnpm dev
```

سيكون التطبيق متاحًا على [http://localhost:3001](http://localhost:3001).
