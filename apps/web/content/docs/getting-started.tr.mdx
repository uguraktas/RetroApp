## Başlarken

Hoş geldiniz! Bu kılavuz, uygulamaya başlamanıza yardımcı olacaktır.

### Ön Gereksinimler

Başlamadan önce aşağıdakilere sahip olduğunuzdan emin olun:

- Node.js 18.x veya üzeri
- npm veya pnpm paket yöneticisi
- React ve Next.js hakkında temel bilgi

### Kurulum

Bağımlılıkları yükleyin:

```bash
pnpm install
```

### Uygulamayı Çalıştırma

Geliştirme sunucusunu başlatın:

```bash
pnpm dev
```

Uygulama [http://localhost:3001](http://localhost:3001) adresinde kullanılabilir olacaktır.
