## Overview

CodeBaseHub is a full-stack SaaS starter kit designed to help developers launch their MVP faster. Built with modern technologies and best practices, it provides everything you need to start building your next big idea.

> **Note:** This starter kit is perfect for developers who want to focus on building features rather than setting up infrastructure.

## Features

### Authentication
Full authentication system with email/password, OAuth, and two-factor authentication powered by Better Auth.

### Multi-language Support
Built-in i18n with support for English, Turkish, and Arabic.

### Beautiful UI
Modern, accessible UI built with shadcn/ui and Tailwind CSS.

### Type-Safe
Full TypeScript support with strict type checking.

### Database Ready
PostgreSQL integration with Drizzle ORM.

### Dark Mode
Beautiful dark mode support out of the box.

## Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/your-org/codebasehub.git
cd codebasehub
```

### 2. Install Dependencies

```bash
pnpm install
```

### 3. Set Up Environment Variables

```bash
cp .env.example .env
```

Update the `.env` file with your configuration.

### 4. Run the Development Server

```bash
pnpm dev
```

Open [http://localhost:3001](http://localhost:3001) in your browser.

## What's Included

- **Next.js 15** - The latest React framework with App Router
- **TypeScript** - Full type safety across the entire stack
- **Better Auth** - Comprehensive authentication solution
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Beautiful, accessible UI components
- **Fumadocs** - This powerful documentation system
- **Drizzle ORM** - Type-safe database queries
- **Monorepo Setup** - Organized with Turborepo
- **Code Quality** - ESLint, Prettier, and Biome configured

> **Warning:** Make sure you have Node.js 18.x or higher and pnpm installed before getting started.
