## التثبيت

اتبع هذه الخطوات لتثبيت التطبيق.

### متطلبات النظام

- **نظام التشغيل**: macOS أو Windows أو Linux
- **Node.js**: الإصدار 18.x أو أعلى
- **مدير الحزم**: pnpm (موصى به)، npm، أو yarn

### الخطوة 1: استنساخ المستودع

```bash
git clone https://github.com/your-org/your-repo.git
cd your-repo
```

### الخطوة 2: تثبيت التبعيات

باستخدام pnpm (موصى به):

```bash
pnpm install
```

أو باستخدام npm:

```bash
npm install
```

### الخطوة 3: إعداد متغيرات البيئة

انسخ ملف البيئة النموذجي:

```bash
cp .env.example .env
```

عدّل ملف `.env` بإعداداتك:

```bash
DATABASE_URL="رابط-قاعدة-البيانات"
NEXT_PUBLIC_API_URL="رابط-api"
```

### الخطوة 4: تشغيل خادم التطوير

```bash
pnpm dev
```

يجب أن يكون تطبيقك الآن يعمل على [http://localhost:3001](http://localhost:3001).

## استكشاف الأخطاء

### المشاكل الشائعة

**المنفذ قيد الاستخدام بالفعل**: إذا كان المنفذ 3001 قيد الاستخدام بالفعل، يمكنك تغييره في `package.json`:

```json
{
  "scripts": {
    "dev": "next dev --port=3002"
  }
}
```

**أخطاء عدم العثور على الوحدة**: جرّب إزالة `node_modules` وإعادة التثبيت:

```bash
rm -rf node_modules
pnpm install
```
