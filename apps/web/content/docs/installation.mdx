## Prerequisites

Before you begin, ensure you have the following installed:

> **Required:** Node.js 18.x or higher and pnpm package manager

### System Requirements

| Requirement | Version | Notes |
|------------|---------|-------|
| **Node.js** | 18.x or higher | [Download](https://nodejs.org) |
| **pnpm** | 8.x or higher | Recommended package manager |
| **PostgreSQL** | 14.x or higher | For database (optional) |
| **Git** | Latest | For version control |

### Install pnpm

```bash
# macOS/Linux
curl -fsSL https://get.pnpm.io/install.sh | sh -

# Windows
iwr https://get.pnpm.io/install.ps1 -useb | iex

# Using npm
npm install -g pnpm
```

## Installation Steps

### 1. Clone the Repository

```bash title="Terminal"
git clone https://github.com/your-org/codebasehub.git
cd codebasehub
```

### 2. Install Dependencies

```bash title="Terminal"
pnpm install
```

This will install all dependencies for the monorepo.

### 3. Set Up Environment Variables

```bash title="Terminal"
cp .env.example .env
```

Edit the `.env` file:

```bash title=".env"
# Database
DATABASE_URL="postgresql://user:password@localhost:5432/codebasehub"

# Authentication
AUTH_SECRET="your-super-secret-key-min-32-chars"
AUTH_TRUST_HOST="true"

# App Configuration
NEXT_PUBLIC_APP_URL="http://localhost:3001"
```

> **Tip:** Generate a strong AUTH_SECRET using: `openssl rand -base64 32`

### 4. Set Up the Database (Optional)

**Using Docker:**

```bash title="Terminal - Docker Setup"
docker run -d \
  --name codebasehub-db \
  -e POSTGRES_PASSWORD=password \
  -e POSTGRES_DB=codebasehub \
  -p 5432:5432 \
  postgres:15-alpine

pnpm db:push
```

**Using Local PostgreSQL:**

```bash title="Terminal - Local PostgreSQL"
# macOS
brew install postgresql@15
brew services start postgresql@15

# Create database
createdb codebasehub

# Run migrations
pnpm db:push
```

### 5. Run the Development Server

```bash title="Terminal"
pnpm dev
```

Open [http://localhost:3001](http://localhost:3001) in your browser.

## Troubleshooting

### Port 3001 is already in use

```bash
# Kill process on port 3001 (macOS/Linux)
lsof -ti:3001 | xargs kill

# Or change the port
pnpm dev --port 3002
```

### Module not found errors

```bash
rm -rf node_modules pnpm-lock.yaml
pnpm install
```

### Database connection errors

Verify your `DATABASE_URL` is correct and the database is running.

### Authentication errors

Make sure `AUTH_SECRET` is set and at least 32 characters long.
