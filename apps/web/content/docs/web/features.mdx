## Authentication & Security

CodeBaseHub includes a complete authentication system powered by **Better Auth**, supporting multiple authentication methods and security features.

<Tabs items={['Email/Password', 'OAuth', 'Two-Factor Auth']}>
  <Tab value="Email/Password">
    **Email/Password Authentication**

    Secure user registration and login with email verification:

    ```typescript title="lib/auth.ts"
    import { authClient } from "@/lib/auth-client";

    // Sign up
    await authClient.signUp.email({
      email: "user@example.com",
      password: "secure-password",
      name: "John Doe",
    });

    // Sign in
    await authClient.signIn.email({
      email: "user@example.com",
      password: "secure-password",
    });
    ```

    Features:
    - Bcrypt password hashing
    - Email verification
    - Password reset flow
    - Session management
  </Tab>

  <Tab value="OAuth">
    **OAuth Authentication**

    Sign in with popular OAuth providers:

    ```typescript title="components/oauth-login.tsx"
    // Sign in with OAuth providers
    await authClient.signIn.social({
      provider: "github",
      callbackURL: "/dashboard",
    });
    ```

    Supported providers:
    - GitHub
    - Google
    - Discord
    - Easy to add more providers
  </Tab>

  <Tab value="Two-Factor Auth">
    **Two-Factor Authentication**

    Enhanced security with TOTP-based 2FA:

    ```typescript
    // Enable 2FA
    await authClient.twoFactor.enable({
      password: "user-password",
    });

    // Verify 2FA code
    await authClient.twoFactor.verify({
      code: "123456",
    });
    ```

    Features:
    - TOTP-based authentication
    - QR code generation
    - Backup codes
    - Optional per user
  </Tab>
</Tabs>

<Callout>
All authentication routes are protected with CSRF tokens and secure session management.
</Callout>

## Internationalization (i18n)

Built-in support for multiple languages with **next-intl**, making it easy to reach a global audience.

<Cards>
  <Card title="Multi-language Support" icon="Globe">
    English, Turkish, and Arabic with easy expansion to more languages.
  </Card>

  <Card title="RTL Support" icon="Code">
    Automatic right-to-left layout for Arabic and other RTL languages.
  </Card>

  <Card title="Type-safe Translations" icon="FileCode">
    Fully typed translations with TypeScript autocomplete.
  </Card>
</Cards>

### Usage Example

<Tabs items={['Component', 'Server', 'Client']}>
  <Tab value="Component">
    ```typescript title="components/welcome.tsx"
    import { useTranslations } from 'next-intl';

    export function MyComponent() {
      const t = useTranslations('common');
      return (
        <div>
          <h1>{t('welcome.title')}</h1>
          <p>{t('welcome.description')}</p>
        </div>
      );
    }
    ```
  </Tab>

  <Tab value="Server">
    ```typescript title="app/[locale]/page.tsx"
    import { getTranslations } from 'next-intl/server';

    export default async function Page() {
      const t = await getTranslations('common');

      return <h1>{t('welcome.title')}</h1>;
    }
    ```
  </Tab>

  <Tab value="Client">
    ```typescript title="components/language-switcher.tsx"
    import { useLocale } from 'next-intl';
    import { useRouter } from 'next/navigation';

    export function LanguageSwitcher() {
      const locale = useLocale();
      const router = useRouter();

      const changeLanguage = (newLocale: string) => {
        router.push(`/${newLocale}`);
      };

      return (
        <select value={locale} onChange={(e) => changeLanguage(e.target.value)}>
          <option value="en">English</option>
          <option value="tr">Türkçe</option>
          <option value="ar">العربية</option>
        </select>
      );
    }
    ```
  </Tab>
</Tabs>

## Database & ORM

PostgreSQL database with **Drizzle ORM** for type-safe database queries.

<Steps>
  <Step>
    ### Define Schema

    ```typescript title="lib/schema.ts"
    import { pgTable, text, timestamp, uuid } from "drizzle-orm/pg-core";

    export const users = pgTable("users", {
      id: uuid("id").defaultRandom().primaryKey(),
      email: text("email").notNull().unique(),
      name: text("name").notNull(),
      createdAt: timestamp("created_at").defaultNow().notNull(),
    });
    ```
  </Step>

  <Step>
    ### Query Data

    ```typescript title="lib/queries.ts"
    import { db } from "@/lib/db";
    import { users } from "@/lib/schema";
    import { eq } from "drizzle-orm";

    const user = await db.query.users.findFirst({
      where: eq(users.email, "user@example.com"),
    });
    ```
  </Step>

  <Step>
    ### Run Migrations

    ```bash
    # Generate migration
    pnpm db:generate

    # Apply migration
    pnpm db:migrate
    ```
  </Step>
</Steps>
