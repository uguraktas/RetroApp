## Web Platform

CodeBaseHub provides a complete web development stack built on Next.js 15 with App Router.

<Callout type="info">
The web platform is the core of CodeBaseHub, offering server-side rendering, static generation, and API routes.
</Callout>

### Key Technologies

<Tabs items={['Next.js', 'React', 'TypeScript']}>
  <Tab value="Next.js">
    **Next.js 15** - The latest version with App Router

    - Server Components by default
    - Streaming and Suspense
    - Built-in optimizations
    - File-based routing
  </Tab>
  <Tab value="React">
    **React 19** - The latest React version

    - Server Components
    - Actions and Forms
    - Enhanced hooks
    - Improved performance
  </Tab>
  <Tab value="TypeScript">
    **TypeScript 5** - Full type safety

    - Strict mode enabled
    - Type inference
    - IntelliSense support
    - Compile-time checks
  </Tab>
</Tabs>

### Getting Started

<Steps>
  <Step>
    ### Install Dependencies

    ```bash
    pnpm install
    ```
  </Step>

  <Step>
    ### Run Development Server

    ```bash
    pnpm dev
    ```
  </Step>

  <Step>
    ### Open in Browser

    Navigate to [http://localhost:3001](http://localhost:3001)
  </Step>
</Steps>
